# Part 1: Getting Started

Welcome to the first part of our Next.js tutorial series. 

In this section, we will cover the basics of the App Router and how it changes the way we build React applications. 

## The App Directory

With Next.js 13+, the `app` directory introduces a new paradigm for routing, layouts, and server components. 
Instead of file-based routing where every file is a page, the app directory uses folders to define routes, and a `page.tsx` file to make that route publicly accessible.
